When the user asks you to "add logs" or "log today's progress", you must update the `/home/empiric/Desktop/Kashyap/nexus/daily-logs.md` file.

**Rules for Logging:**
1. Append the new section at the top of the logs list (directly below the `# Daily Logs` header).
2. Always format the date as a Level 2 Heading using the format: `## [Day] [Month] [Year]` (e.g., `## 4th June 2026`).
3. Write exactly 3 or 4 concise bullet points.
4. Focus only on summarizing the most important architectural changes, features, or bug fixes completed during the session.
